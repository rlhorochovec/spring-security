# spring-security
Projeto utilizando Spring Boot, Spring Security, MySQL e Thymeleaf.

## Dependências
- JDK: 17
- Apache Maven: 3.6.3
- Spring Boot: 2.4.1
- Spring Security: 5.4.2
- Thymeleaf: 3.0.11
- Bootstrap: 4.6.2
- MySQL: 8.0.22

## Instalação
Baixando as dependências

```bash
  cd spring-security
  mvn install
```

## MySQL
Script do banco de dados:
- script/springsecurity.sql

## Acessando
- http://localhost:8080/
- Usuários: admin e user
- Senha: 123456789
